작성 목적

그동안 스프링 부트를 사용하면서 다양한 설정, 쉬운 기능 구현, Annotation을 사용한 간결한 코드 구현등 장점을 많이 보았다. 인터넷을 통해 검색 하면 스프링 관련 정보는 차고 넘치는 실정이지만 필요한 내용을 하나하나 검색 하기도 힘들고 (사실 귀찮은 점이 더 크다 하하..), 실제로 현업을 하다보니 자주 쓰는 기능들이 있어서 이를 하나의 스토리로 묶어서 하나씩 정리 해보기로 했다.

    public static void main(String[] args) {
